Información Bienvenido a la Aplicación de Registro de Asistencia

¡Gracias por descargar nuestra aplicación de registro de asistencia! Esta aplicación está diseñada para simplificar el proceso de seguimiento de la asistencia en diferentes clases.

Paso a Paso: Cómo Utilizar la Aplicación

Sigue estos simples pasos para comenzar a utilizar la aplicación:

  1. Abre la aplicación en tu dispositivo móvil.
  2. Inicia sesión con tu cuenta o crea una nueva cuenta si es necesario.
  3. Explora la pantalla principal para ver los eventos o clases disponibles.
  4. Selecciona el evento o clase en la que deseas registrar la asistencia.
  5. Escanea el código QR proporcionado en el lugar del evento utilizando la cámara de tu dispositivo.
  6. ¡Listo! Has registrado la asistencia exitosamente.
